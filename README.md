# 🏏 Sri Lanka Cricket Ticket Booking System

## 📌 Project Overview
This project is a full-stack web application designed to simulate the official Sri Lanka Cricket ticket booking experience. It features a blue and gold themed UI, real match data stored in MongoDB, an interactive stadium seating map, and a complete booking system with reference generation.

---

## ✨ Features
🏠 Homepage — SLC styled landing page  
🏟 Match Listings — Upcoming matches from database  
🗺 Interactive Stadium Map — Clickable seating sections  
🎟 Ticket Booking — Select seats and confirm  
📋 Booking Reference — Auto-generated ID  
💾 MongoDB Database — Stores all data  
📉 Seat Availability — Updates after booking  
🌱 Admin Seed Route — Add sample matches  

---

## 🛠 Technology Stack
Frontend: HTML, CSS, JavaScript  
Backend: Node.js, Express.js  
Database: MongoDB  
ODM: Mongoose  
Tools: VS Code, MongoDB Compass

## ⚙️ Getting Started
1. Clone the repository  
2. Run npm install  
3. Start MongoDB  
4. Run npm start  
5. Open index.html
